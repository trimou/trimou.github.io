Trimou

Trimou is a Mustache implementation written in Java. Since version 1.5.0 Trimou has a helper API inspired by Handlebars. The goal is to provide a simple to use and easy to extend templating engine for any Java SE or Java EE application. So far there are five ready-to-use extensions which provide integration with CDI, Servlets, PrettyTime, HtmlCompressor and google-gson.

Don't forget to check the example web application .
